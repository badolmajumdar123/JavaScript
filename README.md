# ⚡ JavaScript Basics Learning Project

## 🎯 About JavaScript
JavaScript is a:
- High-level programming language  
- Interpreted language  
- Object-based / Object-oriented language  

It is mainly used for Web Development (Frontend + Backend with Node.js).

---

## 📖 Project Description
This project is created for learning JavaScript basics to advanced concepts such as:
- Variables  
- Data Types  
- Functions  
- Loops  
- Objects & Arrays  

---

## 🧠 JavaScript Data Types Covered

### 🟢 Basic Types
- Number → 10, 10.5  
- String → "Hello"  
- Boolean → true / false  
- Undefined → undefined  
- Null → null  

---

### 🟡 Intermediate Types
- Array → [1, 2, 3]  
- Object → { name: "Badol" }  

---

### 🔴 Advanced Types
- Function → function() {}  
- Promise → async operations  
- Symbol → unique value  
- BigInt → large numbers
